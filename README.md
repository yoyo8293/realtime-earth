# 🌍 Realtime Activity Earth | 实时活动地球

一个基于 Three.js 的实时3D地球可视化项目，展示全球活动节点和数据流动效果。

## ✨ 特性

- 🌏 **3D地球渲染** - 高精度地球模型，带海洋和大陆纹理
- ☁️ **动态云层** - 逼真的云层效果，独立旋转
- 🌟 **大气光晕** - Shader实现的蓝色大气光晕效果
- ✨ **粒子系统** - 环绕地球的动态粒子效果
- 📍 **活动标记** - 全球活动节点标记，带脉冲动画
- 🎮 **交互控制** - 鼠标悬停显示节点信息，点击交互
- 📊 **实时数据** - 动态更新的统计数据

## 🚀 快速开始

### 方式1：直接打开
在浏览器中打开 `index.html` 文件即可。

### 方式2：本地服务器（推荐）
```bash
# 进入项目目录
cd /root/clawd/projects/realtime-earth

# Python 3
python3 -m http.server 8080

# 或 Node.js
npx serve

# 然后访问 http://localhost:8080
```

## 🎮 操作说明

| 操作 | 说明 |
|------|------|
| 鼠标移动 | 悬停查看节点详情 |
| 鼠标点击 | 选中节点 |
| ⟲ 按钮 | 开启/停止自转 |
| ✦ 按钮 | 显示/隐藏粒子 |
| ⌖ 按钮 | 重置视角 |

## 🛠️ 技术栈

- **Three.js** - 3D渲染引擎
- **HTML5 Canvas** - 动态纹理生成
- **GLSL Shaders** - 大气光晕效果
- **Vanilla JavaScript** - 无框架依赖

## 🌐 演示数据

项目包含模拟的全球活动节点：
